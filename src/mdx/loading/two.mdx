It is automatically activated if you <a data-astro-reload href="/library/Installation/">install astro-vtbot</a> as an astro integration.

The default icon that is displayed is the favicon of the current page. 
Of course, you can tweak the look & feel of the <a data-astro-reload href="/library/LoadingIndicator/">&lt;LoadingIndicator/> component</a> to your liking.

If you want to know what happens behind the scenes, here is a rough sketch of the code that can be used to display a loading indicator if you do not use the reusable `<LoadingIndicator/>` component:

```javascript title="loading.js"
document.addEventListener('astro:before-preparation', beforePreparation);
document.addEventListener('astro:after-preparation', afterPreparation);

function beforePreparation() {
	document.querySelector('#loading').classList.add('show');
}
function afterPreparation() {
	document.querySelector('#loading').classList.remove('show');
}
```
